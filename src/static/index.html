How to use

Anubis/Go-away: You should find the challenge JSON in `head > script#anubis_challenge` or a network request that ends with `/make-challenge`. Copy the challenge JSON and paste it into the textarea below. Characters before the first `{` and after the last `}` are ignored.
Response will be a single redirect URL provided in JavaScript format mirroring the official solver's behavior after a solution is found.

Expected Formats

  • Anubis: {"rules":{"algorithm":"fast/slow","difficulty":4,"report_as":4},"challenge":"xxxxxx"}
  • Go-away: {"challenge":"xxxxxx", "target": "000fff", "difficulty": 16}


NOTICE: this is a public service limited to manageable load and limited concurrency, please do not abuse it. Run it on your own server if you need more resources. Source Code
